--- layout: post title: The World Beyond UI published: 1 ---

Joel Spolsky's latest article, It's Not Just Usability, makes the case that software and websites must aim for a goal far beyond an easy-to-use user interface - a thriving, well-done, and inviting social interface. Projects that master the social interface (such as eBay) can always be improved through better UI design. However, if the social interface is missing, UI tweaks alone are not the answer.
